## 三、检漏仪校准参数（6 个）

| 占位符        | 含义                                                    | 物理意义                        | 单位          |
| ---------- | ----------------------------------------------------- | --------------------------- | ----------- |
| `{{Q0}}`   | 标准漏孔校准漏率 (Calibrated Leak Rate of Standard Leak)      | 标准漏孔在标定条件下的已知漏率值            | **Pa·m³/s** |
| `{{QT}}`   | 测试温度下标准漏孔的漏率 (Standard Leak Rate at Test Temperature) | Q0 经温度修正后的值，用于消除温度对漏率的影响    | **Pa·m³/s** |
| `{{I0}}`   | 检漏仪背景信号 (Leak Detector Background Signal)             | 检漏仪在未引入氦气时的固有输出信号（本质噪声+残余氦） | 分度/div      |
| `{{In}}`   | 检漏仪背景噪声 (Leak Detector Background Noise)              | 背景信号的波动幅度，决定最小可分辨信号         | 分度/div      |
| `{{I}}`    | 泄漏信号 (Leak Signal)                                    | 引入标准漏孔后检漏仪的输出信号             | 分度/div      |
| `{{Qmin}}` | 检漏仪灵敏度 (Leak Detector Sensitivity)                    | 检漏仪本身能明确检测到的最小漏率            | **Pa·m³/s** |

> **Q/I 命名规律**: `Q` 系列 = 漏率 (Pa·m³/s)；`I` 系列 = 检漏仪电流/信号读数 (div)。Qmin 的计算关系：Qmin = Q0 × In / (I - I0)

## 四、测试系统记录（8 个）

| 占位符         | 含义                                       | 物理意义                          | 单位          |
| ----------- | ---------------------------------------- | ----------------------------- | ----------- |
| `{{M0}}`    | 测试系统背景信号 (Test System Background Signal) | 连接被测件后、充氦前，检漏仪的系统总背景读数        | 分度/div      |
| `{{Mn}}`    | 测试系统背景噪声 (Test System Background Noise)  | 系统总背景信号的波动幅度                  | 分度/div      |
| `{{M1}}`    | 泄漏信号 (Leak Signal)                       | 充氦后检漏仪的输出读数（反映组件泄漏+系统背景）      | 分度/div      |
| `{{TS}}`    | 响应时间 (Response Time)                     | 从氦气到达漏孔到检漏仪信号稳定的时间间隔          | **秒(s)**    |
| `{{Qemin}}` | 测试系统灵敏度 (Test System Sensitivity)        | 该测试系统（检漏仪+管路+被测件）实际能达到的最小可检漏率 | **Pa·m³/s** |
| `{{M2}}`    | 喷氦后泄漏信号 (Leak Signal After Helium Spray) | 对局部喷氦后检漏仪的输出读数（用于漏点定位）        | 分度/div      |
| `{{TG}}`    | 氦浓度 (Helium Concentration)               | 罩内或喷氦气流的氦气体积百分比浓度             | **%**       |
| `{{Qs}}`    | 系统测试漏率 (System Measured Leak Rate)       | 经浓度修正后，被测件的实际总漏率              | **Pa·m³/s** |

> **M 序列含义**: M0=背景, M1=充氦后信号, M2=喷氦后信号。Qs 的核心计算公式：Qs = Qemin × (M1 - M0)

## 五、结果判定（2 个）

| 占位符 | 含义 | 取值 |
|---|---|---|
| `{{Qcompare}}` | 实际漏率与验收标准的大小关系 | `greater than`（大于）或 `less than`（小于等于） |
| `{{Qresult}}` | 最终测试结论 | `Pass`（合格）或 `Fail`（不合格） |

**判定逻辑**：当实际漏率 Qs ≤ 验收标准（Acceptance Criteria）时，Qcompare = "less than"，Qresult = "Pass"；反之 Qs > Acceptance Criteria 时，Qcompare = "greater than"，Qresult = "Fail"。

```
基本信息（自然语言命名）:
  Report No. / Contract No / Product Name / Serial No. / Temperature
  Test Date / Relative Humidity / Inspection Area / Test Procedure No. / Acceptance

设备表（{设备}{属性} 模式）:
  LD / SL / T / HC  ×  M / N / VP → 12个设备占位符

检漏仪校准:
  Q0 ──(温度修正)──→ QT         漏率线
  I0, In ──→ I ──→ Qmin         信号→灵敏度线

测试系统:
  M0, Mn ──→ M1 ──→ Qemin ──→ Qs    背景→信号→灵敏度→漏率
                       TS, TG           响应时间, 浓度修正
  M2 (喷氦定位用)

结果:
  Qs vs Acceptance → Qcompare, Qresult
```
